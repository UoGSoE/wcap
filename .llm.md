## Purpose

WCAP is a two-week planning tool that helps IT teams capture who is working, what they’re focusing on, and where they’ll be located, with live updates and quick planning helpers. The primary users are staff (for updating plans) and managers (for visibility and reporting on team/service coverage). It solves the problem of keeping plans up to date and providing clear coverage across teams and services.

## Features (from tests)

- Profile page loads with existing user defaults and allows updates to those defaults
- Profile allows empty defaults and saving profile updates persists changes
- Home page renders with the expected 14-day view starting on Monday
- Saving new entries creates corresponding database records
- Saving updates to existing entries modifies the stored records accordingly
- Copy next copies an entry to the next day only
- Copy rest copies an entry to all remaining days
- Existing entries load their is_available state correctly on mount
- New entries use user defaults to prefill fields
- Existing entries override user defaults when appropriate
- Coverage tab displays a location coverage grid for planning
- Unavailable users show as away in the My Team view
- Unavailable users do not appear in by location view
- Unavailable users do not affect coverage counts
- Team filtering by specific team shows only members of that team
- Team filtering by multiple teams shows all members from those teams
- Admin can download the team report as an Excel file
